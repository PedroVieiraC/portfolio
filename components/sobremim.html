Sobre Mim

Sou desenvolvedor back-end apaixonado por tecnologia e programação. Formado em Sistemas de Informação, tenho experiência sólida em desenvolvimento de aplicações robustas e escaláveis. Minha jornada na programação começou com curiosidade e se transformou em uma paixão genuína por resolver problemas complexos através do código.

Trabalho principalmente com Java e Spring Boot, desenvolvendo APIs RESTful e microserviços. Tenho conhecimento em bancos de dados relacionais (MySQL, PostgreSQL) e não relacionais (MongoDB), além de experiência com versionamento de código usando Git e metodologias ágeis.

Sempre busco estar atualizado com as melhores práticas de desenvolvimento, clean code e arquitetura de software. Acredito que a programação é uma ferramenta poderosa para transformar ideias em soluções reais que impactam positivamente a vida das pessoas.
